# Push this project to your GitHub

Everything is committed locally. Do these 2 steps:

## Step 1: Create the repository on GitHub

Open this link (it pre-fills the repo name):

**https://github.com/new?name=employee-management-app**

- Leave "Add a README" **unchecked** (we already have one).
- Click **Create repository**.

## Step 2: Push from your PC

In a terminal, from this folder (`employee-management-app`), run:

```bash
git push -u origin main
```

If GitHub asks for login, use your username **sumanthrai123** and a **Personal Access Token** (not your password).  
To create a token: GitHub → Settings → Developer settings → Personal access tokens → Generate new token.

---

After the push, your app will be at: **https://github.com/sumanthrai123/employee-management-app**

Others can clone and run it by following the README.
